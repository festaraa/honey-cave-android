# Honey Cave - Development Notes

## Основна Архитектура

### Персонаж (Player)
- Движи се със стрелките
- Скорост: 200 пиксела/секунда
- Съхранява брой активирани чекпойнти

### Чекпойнт (Checkpoint)
- Кога герой ги докосне, те се активират (стават жълти)
- Могат да бъдат активирани само веднъж
- Съобщават на герой че са активирани

### Врата (Door)
- Изисква N активирани чекпойнта (по подразбиране 2)
- Когато се отвори, става зелена
- Проверява брой на активирани чекпойнти при контакт

## Как да разширим игата:

### 1. Добавяне на нови нива:
```
Créare ново Checkpoint или Door в Main.tscn
```

### 2. Промяна на параметри:
- `Player.gd` - `export var speed = 200` (променя скоростта)
- `Door.gd` - `export var required_checkpoints = 2` (променя брой на чекпойнти)

### 3. Добавяне на врага:
Создай новия Enemy.gd със AI

### 4. Добавяне на графика:
Замени ColorRect със Sprite nodes

## Файлове за редактиране:

- `scenes/Main.tscn` - Редактирай места на чекпойнти
- `scenes/Player.gd` - Променяй движение
- `scenes/Checkpoint.gd` - Променяй поведение
- `scenes/Door.gd` - Променяй врата
