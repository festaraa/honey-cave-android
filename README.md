# Honey Cave - Android Game
## Основна Игра

Оригинална Android игра, вдъхновена от класическия геймплей на мобилни игри.

### Как се играе:

1. **Управление**: Използвай стрелките за движение
2. **Активирай чекпойнти**: Докосни червените кръгове (чекпойнти)
3. **Отвори врата**: Активирай необходимия брой чекпойнти, за да отвориш врата (сива)
4. **Премести се напред**: Кога врата е отворена, можеш да напредваш

### Структура на проекта:

```
honey-cave-android/
├── scenes/
│   ├── Main.tscn       # Главна сцена
│   ├── Player.tscn     # Герой (син квадрат)
│   ├── Checkpoint.tscn # Чекпойнти (червени кръгове)
│   ├── Door.tscn       # Врата (сива)
│   ├── Player.gd       # Скрипт на герой
│   ├── Checkpoint.gd   # Скрипт на чекпойнт
│   └── Door.gd         # Скрипт на врата
├── assets/             # Графика (ще добавим)
├── project.godot       # Конфигурация
└── README.md           # Този файл
```

### Какво е готово сега:

- ✅ Базов герой със движение (стрелки)
- ✅ 3 чекпойнта (червени кръгове)
- ✅ Врата която се отваря при 2+ активирани чекпойнта
- ✅ Система за следене на активирани чекпойнти

### Как да тестираш:

1. Отвори този проект в Godot 3.4+
2. Натисни F5 или "Play"
3. Движи се със стрелките
4. Докосни чекпойнтите
5. Активирай 2+ чекпойнта и докосни врата
